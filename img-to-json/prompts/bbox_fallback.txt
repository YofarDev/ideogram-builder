You are an expert image analyst. Below is a list of objects visible in the image that an automated segmentation model could not locate. For each object, provide its bounding box — the tight rectangular region where the object appears.

Output format — return ONLY this JSON structure:

{
  "objects": [
    {
      "name": "vase",
      "bbox": [420, 580, 850, 920]
    },
    {
      "name": "lamp",
      "bbox": [120, 340, 280, 620]
    }
  ]
}

bbox is [{bbox_desc}] — four integers 0-1000, normalized image coordinates.
Origin is top-left: y=0 is the top edge, x=0 is the left edge, y=1000 is the bottom, x=1000 is the right.

Rules:
- One entry per object in the list below. Use the exact name.
- If an object is genuinely not visible, set bbox to null.
- Boxes must tightly enclose the object.

Return ONLY the JSON object. No explanation, no markdown code fences.
