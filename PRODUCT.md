# Product

## Register

product

## Users

AI artists and prompt engineers using Ideogram4 for image generation. They work in focused creative sessions — iterating on prompts, adjusting bounding box layouts, fine-tuning color palettes. They know their tools and expect professional-grade control without hand-holding. Context: desktop-first, likely multi-monitor, ambient light varies (studio or home office).

## Product Purpose

Visual prompt builder that turns creative intent into structured Ideogram4 JSON. Canvas-based bounding box editor lets users place, resize, and label regions. Color palettes, mode toggles, and form fields shape the generation parameters. PNG metadata import, AI prompt enhancement, and ComfyUI/RunPod generation close the loop from idea to image.

Success = users can go from a rough idea to a precise, reproducible JSON prompt faster than hand-editing JSON, with visual feedback at every step.

## Brand Personality

Precise, warm, expert. Like a well-made darkroom — the space itself is dark and warm-toned, but the controls are exact and the workflow is confident. No flash, no decoration for its own sake. The tool earns trust through competence.

## Anti-references

- Generic SaaS dashboards with cream/white backgrounds and purple accents
- Toy-like creative tools with rounded everything and playful illustrations
- AI chatbot interfaces with conversational UI patterns
- Overly minimal monochrome tools that feel cold or sterile

## Design Principles

1. **Canvas is king.** Every UI element exists to serve the canvas workspace. Panels, controls, and toolbars are supporting actors — visible when needed, invisible when not.
2. **Expert confidence.** Trust the user. No tooltips on every control, no onboarding wizards, no hand-holding. Clear labels, predictable layouts, keyboard shortcuts where they matter.
3. **Warm precision.** Dark surfaces with warm undertones. Gold accent carries focus without shouting. The palette feels like a darkroom, not a cockpit — creative ambient, not surveillance.
4. **Iterative flow.** The tool supports rapid iteration. Every action should feel reversible, every state inspectable. Prompt → JSON → generate → import → refine is a loop, not a pipeline.
5. **Structured freedom.** Bounding boxes, color palettes, and form fields provide structure. Free-text fields and AI enhancement provide freedom. Neither dominates.

## Accessibility & Inclusion

WCAG AA compliance. Gold-on-dark palette already provides strong contrast. Focus states visible on all interactive elements. Reduced-motion respected for entrance animations. No known special accommodations required.
